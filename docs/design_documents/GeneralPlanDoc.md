## 开发阶段概览

| 阶段 | 名称 | 预计工期 | 核心目标 |
|------|------|----------|----------|
| Phase 1 | 核心框架 | 4-6周 | 基础移动、收集、生存循环 |
| Phase 2 | 基地系统 | 4-6周 | 气密性、氧气系统、扩建 |
| Phase 3 | 资源与进程 | 3-4周 | 科技树、燃料系统、稀有资源 |
| Phase 4 | 威胁系统 | 3-4周 | 陨石撞击、减压、环境危害 |
| Phase 5 | 世界生成 | 4-5周 | 小行星带、区域、动态生成 |
| Phase 6 | 叙事内容 | 3-4周 | 废弃遗迹、环境叙事、信号系统 |
| Phase 7 | 打磨与优化 | 2-3周 | 音效、视觉、性能优化 |

## 关键路径

**最小可玩版本 (MVP) 需完成:**
1. Milestone 1.1 - 1.5 (核心框架)
2. Milestone 2.1 - 2.5 (基地系统基础)
3. Milestone 3.1 (燃料)
4. Milestone 4.1 (陨石威胁)
5. Milestone 5.1 - 5.2 (基础世界)

**预计 MVP 工期:** 12-16 周

## 风险与缓解

| 风险 | 影响 | 缓解措施 |
|------|------|----------|
| 气密性算法复杂 | 高 | 早期原型验证，备选简化方案 |
| 性能问题 | 中 | 持续性能监控，对象池化 |
| 内容量不足 | 中 | 程序化生成，模块化设计 |
| 叙事吸引力 | 低 | 保持可选性，专注核心玩法 |
