# 《Look at the sky》——游戏设计文档

## 1. 核心概念与设计支柱

**核心概念：**
《Look at the sky》是一款第一人称视角的太空生存制作游戏，玩家被困在太空中的一个逃生舱上。玩家仅靠一个镐子为工具，必须通过收集漂浮物生存下来，扩建他们的漂浮避难所，并最终揭开这个远在木星的边缘世界的神秘面纱。

**核心设计支柱：**

1.  **核心机制:** 游戏的核心机制是使用镐子破坏周围飘浮的陨石。它是驱动资源获取、风险/回报决策以及漂泊感的基础循环。
2.  **扩建：** 玩家的进步直接且物理地体现在木筏的成长上——从一个小方块发展为多层漂浮的堡垒。**气密性系统**将扩建从一个单纯的建造行为提升为生存策略的核心决策点。
3.  **孤寂与探索并存：** 游戏平衡了广阔太空带来的宁静、孤独感与在故事小行星上紧张、专注的发现感。这营造出一种由放松到紧张的强有力的节奏变化。
4.  **有引导的自由：** 玩家可以自由地在任何地方建造和航行，但叙事进程受到其科技发展的限制，在这个看似无边的世界中为玩家提供了清晰的目标感和方向感。

## 2. 核心玩法机制与设计原理

### 2.1. 镐子与收集循环
- **机制：** 玩家的主要工具是一个用来破坏陨石的镐子。
    - **核心反馈循环：** 收集 -> 建造/燃料 -> 生存（氧气） -> 航行至更多漂浮物。这个循环简单易懂，却提供了一个令人满足、永无止境的进步循环。
- **设计原理：** 镐子不仅是工具，更是玩家与世界的唯一直接交互点。每一次挥镐都是一次决策：这块陨石是否值得我消耗燃料靠近？它是否会撞击我的基地？这种持续的微决策构成了游戏体验的基石。

### 2.2. 作为移动基地的逃生舱——气密性与生命维持
- **机制：**
    - **生命维持双轨制：** 玩家需要同时管理**氧气**和**燃料**。
        - **氧气制造器：** 开局自带的核心设备。消耗**燃料**（从陨石中提取）来为基地内所有**气密空间**填充氧气。
        - **个人氧气储备：** 离开基地后，玩家仅靠宇航服内的氧气存活，时间以分钟计算（类似《Outer Wilds》）。这创造了强烈的探索紧迫感。
        - **燃料：** 既是氧气来源，也是**宇航服推进器**和后期**火箭发动机**的能量来源。
    - **基于网格的气密建造系统：**
        - 玩家可以放置地基、墙壁、地板、天花板和楼梯来扩建。
        - **气密性逻辑：** 一个封闭的空间（由墙壁和气闸舱构成）会自动被氧气制造器充氧。如果外壳被陨石击穿，该空间会瞬间减压，所有氧气逸散。
        - **气闸舱：** 科技树中的关键节点。允许玩家进出基地而不泄露整个基地的氧气。
- **设计原理：**
    - **进步的物理体现：** 你的成功不仅体现在基地的大小，更体现在它的**复杂性**。一个多层、多舱室、带观察室的基地不仅是视觉上的成就，更是你在气密性设计上成熟的证明。
    - **陨石作为调节器：** 陨石是动态的、不可预测的威胁。它替代了《Raft》中的鲨鱼，迫使玩家不断巡逻、监听撞击声、迅速修补漏洞。它惩罚“粗放式扩张”——一个远离核心的、未受保护的舱室被击穿，可能导致整个基地失氧。
    - **导航与移动的演变：**
        - **早期（惯性滑行）：** 玩家只能依靠初始动量滑行，或使用宇航服的微量推进调整方向。这强化了太空的广阔与无助感。
        - **中期（引擎解锁）：** 获得火箭发动机后，玩家从漂流者变为航海家。可以主动探索更远的区域，但燃料管理成为新的核心挑战。
        - **气密性作为新手引导：** 解锁“气阀”的需求迫使玩家学习游戏的核心生存逻辑——封闭空间等于安全。它自然地引导玩家从“随意搭建”过渡到“策略性规划”。

### 2.3. 进程与科技树
- **机制：** 游戏进程与研究桌绑定。玩家研究从陨石或废弃空间站中提取的稀有样本，解锁新的蓝图：从高效的氧气制造器、气阀，到火箭发动机和高级生活模块。
- **设计原理：**
    - **限制探索范围：** 没有引擎，你只能被惯性带向远方，无法回头。没有高效氧气系统，你无法在远离基地的深空长时间探索废弃空间站。科技树自然划定了游戏的章节。
    - **有意义的发现：** 每一块陨石都可能含有珍稀矿物或外星微生物样本。这些不仅是燃料，更是解锁新科技、推动剧情的关键。这赋予了收集循环更深层的价值。

### 2.4. 威胁与生存平衡
- **机制：**
    - **氧气与燃料：** 核心生存指标。氧气是短期计时器，燃料是长期战略资源。
    - **陨石撞击：** 持续的、随机的外部威胁。撞击会产生独特的音效，提醒玩家必须迅速穿上宇航服、外出修补。
    - **减压风险：** 如果你在修补漏洞前进入受损舱室，会被瞬间吸入太空。如果你在建造时忘记安装气阀，整个基地可能因一次错误开门而失氧。
    - **环境危害：** 废弃空间站中的辐射泄漏、失温、以及可能的太空生物。
- **设计原理：**
    - **资源消耗点：** 燃料是驱动一切的货币。它让你呼吸、让你移动、让你扩张。这使收集循环永不停歇。
    - **太空行走的风险/回报：** 离开基地永远是高风险行为。氧气计时器在倒计时，陨石可能随时袭来，而你必须在有限时间内完成任务并返回气闸舱。
    - **节奏把控：** 在基地内是平静的维护与规划（检查氧气储量、规划扩建）。陨石撞击和太空行走是高强度的紧张时刻。这种切换防止游戏陷入单调，创造了“安全—危险—安全”的情感波动。

## 3. 世界与叙事设计

### 3.1. 作为角色的世界
- **设计：** 世界是木星边缘的一片小行星带，拥有不同的区域：
    - **稀疏带：** 空旷、宁静，只有零星的陨石。这里是新手区，也是冥想时刻的发生地。
    - **密集带：** 陨石如雨，航行风险极高，但资源丰富。这里是风险/回报决策的核心区域。
    - **废弃遗迹：** 前人类探索任务的残骸——废弃的空间站、碎裂的殖民舱、巨大的货船残骸。这些是主要的故事区域，内部充满辐射、陷阱和线索。
- **原理：** 太空不是空洞的。它的密度、颜色、背景星辰和远方木星的巨大身影，都在讲述一个关于孤独与渺小的故事。发现一个废弃空间站，就像在沙漠中发现一座金字塔——它既是避难所，也是坟墓。

### 3.2. 通过环境叙事
- **设计：** 主线故事是可选的。玩家通过破解来自深空的微弱信号，定位到大型剧情遗迹。在这些残骸中，他们可以找到宇航员的日志、实验记录、求救信号，拼凑出木星基地的历史和自我毁灭的原因。
- **原理：**
    - **保留玩家自由度：** 只想建造和生存的玩家可以永远留在小行星带。而寻求目标的玩家则有清晰的方向——追随信号，直到木星的阴影深处。
    - **情感回报：** 你安全的、充氧的基地，与那些漂浮在真空中数十年的尸体形成强烈对比。玩家不仅是生存者，也是收尸人、考古学家和最后的听众。

## 4. 玩家心理与体验

### 4.1. 从遇难者到太空旅者的旅程
- **游戏早期：** 玩家是被动的。逃生舱狭小，氧气稀缺，陨石撞击令人心惊胆战。核心感受是**脆弱与焦虑**。每一次太空行走都是一次赌博。
- **游戏中期：** 玩家成为基地的工程师。拥有了气阀、稳定的氧气供应和引擎。可以自信地规划扩建，探索密集带。感受转变为**谨慎的掌控感**和**对未知的渴望**。
- **游戏后期：** 玩家成为小行星带的主人。基地是一座移动的太空堡垒，可以前往任何区域，直面废弃遗迹最深处的秘密。感受是**使命、力量与终结**——以及当真相揭晓时，可能再次袭来的脆弱。

### 4.2. 核心情感节奏
- **平静/冥想状态：** 在充氧的观察室里看着木星缓缓旋转，听着自己呼吸的回音，整理存储箱，规划下一次扩建。这是游戏的默认状态，提供一种近乎冥想的孤独美感。
- **焦虑/紧张感：** “咚”——陨石击穿舱壁的沉闷响声。氧气警报响起。在废弃空间站中，手电筒照向黑暗走廊的尽头。这些时刻打破了平静，激活了求生本能。
- **成就感/喜悦：** 成功密封第一个大型舱室。第一次启动引擎，让基地朝着自己选择的方向前进。找到关键蓝图或日志，揭开幕后黑手的一角。这些是克服挑战后的情感奖励。
